### 📂 Archivos de Proyecto: NexusQuant

[NexusQuant](https://nexusquant.vercel.app/) es mi desarrollo insignia: un ecosistema SaaS profesional que centraliza el análisis de estrategias y la creación de portafolios de trading institucionales.

*   **Simulador Web MT5:** Motor de backtesting 100% nativo en la web, diseñado para almacenar y procesar históricos de velas sin depender de software de escritorio.
*   **Laboratorio de Optimización:** Arquitectura de flujos de trabajo institucionales para análisis `In-Sample` ➔ `Forward` ➔ `Cruce Walk-Forward`.
*   **Análisis Predictivo:** Desarrollo de paneles interactivos para Simulaciones de Montecarlo y Análisis Estacional.
*   **Distribución de Capital:** Algoritmos de cálculo de capital teórico con interfaces de exportación de datos paramétricos (CSV).
*   **Lógica de Negocio:** Diseño de modelos de monetización escalonada (Freemium/Premium) gestionando límites operativos para bots y pares transaccionales.
